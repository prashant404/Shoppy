 An E-commerce site build using React, Redux, Redux-toolkit, React-router-dom and Typescript
## Built with

+ React
+ Redux
+ Redux toolkit
+ Redux-thunk
+ Typescript
+ Axios
+ React Router v6
+ Framer Motion
+ SwiperJS

## Features

+ Fully responsive design
+ Add to cart
+ Delete from cart
+ Checkout
+ Lazy loading of images
+ Image Optimization
+ Static footer containing a set of links
+ Quick View
+ Product Details
+ Cart
+ Login

## Installation

- [ ] Clone the Repository
```
https://github.com/athulrajhere/FlowyCart-ecommerce-react-redux.git
```
- [ ] Enter into the directory
```
cd FlowyCart-ecommerce-react-redux
```
- [ ] Install all dependencies
```
npm install
```
- [ ] Start the Development Server
```
npm run dev
```
